# 车牌号专属键盘
自定义车牌号键盘，满足常见车牌和新能源车牌需求，可任意修改车牌号  
## 使用示例：
```
<plate-number bind:plate="onPlate"></plate-number>
```

 ## 事件规则：
 
  名称 | 说明
  ---- | ----
  onPlate | 键盘点击操作的结果值
